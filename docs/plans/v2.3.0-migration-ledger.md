# Elves 2.3 migration ledger (temporary, proof-oriented)

This ledger inventories normative 2.2 requirements and records whether each is
**retained**, **changed**, or **retired** in 2.3. The machine-readable source of
truth is `scripts/cobbler_runtime/canonical_contract.py::MIGRATION_LEDGER`.

Do not treat this file as a second protocol manual. Delete or archive it after
the 2.3 release closes if the code ledger and tests already carry the proof.

## How to read

| Column | Meaning |
|--------|---------|
| Disposition | retained / changed / retired |
| Old location | 2.2 surface |
| New location | 2.3 surface |
| Proof | focused test module or fixture |
